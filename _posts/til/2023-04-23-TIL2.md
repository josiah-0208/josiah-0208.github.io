## 1. npm run이 default이다.
npm init로 생성된 node 프로젝트에서 npm run은 package.json의 script를 실행하는 명령어이다.  
npm start의 경우는 npm run start가 맞는 형태이지만, 자주 사용하는 script의 경우 생략이 가능하다. 반대로, run을 생략하면 안되는 경우도 있다.
